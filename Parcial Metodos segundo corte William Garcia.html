Ejercicio 1 — RK3, EDO de orden 3
$$y''' - ny'' = nx^n - n^2$$
$$y(0) = y'(0) = y''(0) = n$$
Sistema: $y_1'=y_2,\quad y_2'=y_3,\quad y_3' = ny_3 + nx^n - n^2$
Fórmulas RK3 — sistema de 3 ecuaciones
$k1_i$
$$k1_i = f_i(x_n,\,Y_n)$$
$k2_i$
$$k2_i = f_i\!\left(x_n+\tfrac{h}{2},\;Y_n+\tfrac{h}{2}k1\right)$$
$k3_i$
$$k3_i = f_i\!\left(x_n+h,\;Y_n - hk1 + 2hk2\right)$$
$y_{i,n+1}$
$$y_{i,n+1} = y_{i,n} + \frac{h}{6}(k1_i + 4k2_i + k3_i)$$
Parámetros
Condiciones: $y_1(0)=y_2(0)=y_3(0)=n$
Tabla de resultados
Gráficas individuales por variable
$y_1 = y(x)$
$y_2 = y'(x)$
$y_3 = y''(x)$
Pendientes $k$ por iteración
Gráfica global — $y_1(x)$, $y_2(x)$, $y_3(x)$
